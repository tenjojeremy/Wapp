## Installation

```
yarn add @tenjojeremy/wapp
```

## .wapp.manifest.js

### authentication

Options: `firebase`

```md
authentication: 'firebase',
```

### Firebase

```md
firebase: {
config: firebaseConfig,
},
```

### Fonts

defaults: source: 'google'

```md
typogrpahy: {
source = 'firebase
fonts: ['roboto', { name: 'montserrat', weights: [500] }],
},
```
